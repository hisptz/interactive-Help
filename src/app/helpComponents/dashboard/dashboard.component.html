Setting up the dashboard

Setting the dashboard this is done by selection any favorite recent dashboard available on the top menu row under search text in put field. Where this is achieved by clicking just your favorite dashboard or recent used dashboard to make it active for use and display

Creating new DashBoard

This is done by doing the following steps:

  1. Click Add button on top left row on current active dashboard.

  2. Add components by searching from the search area in the middle section of the dashboard. This components may be saved charts, pivot tables, reports, GIS reports etc.

  3. After search result found and displayed, then click add button on the right side of particular item to be added to the dashboard

Manage dashboard components

Dashboard components are easy to use, where by they help user manage the added item to the dashboard. This items are Resize, Remove, Explore and Share interpretation

  1. Resize: This option expands your selected card for better view also strectsh back to default size when its clicked again.

  2. Remove: This option when clicked it remove the selected component from dashboard. Its also useful in removing duplicated components.

  3. Explore: This option is useful on times when you need to edit the desired component from dashboard, so when explore option is clicked it opens the particular application of the component on the dashboard. Thus when it gives you abilities to edit the chart or table or report

Manage dashboard

In dashboard management there are options to which helps you manage the current dashboard, and this options are Rename, message, delete, translate

  1. Rename: This option gives user the choice to edit the current dashboard name to a new name desired.

  2. Message: This option when clicked adds the message component to the current dashboard, so as to quick access the messages.

  3. Delete: This option when clicked it deletes the current dashboard from your dashboard lists.

  4. Translate: Its very useful option that translates your current dashboard to user chosen language of preference.
